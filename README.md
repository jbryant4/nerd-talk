# Nerd-talk
  
A NoSQL database used to make the backend of a social media application.

## Installation
Follow these steps to install and use code:
- clone repository
-  open terminal
-  npm install 
-  npm start 
-  open insomnia to test all crud routes(ref walk-through video for the crud routes)
